# 设备角色

设备可以根据功能角色进行组织，这些角色由用户完全自定义。例如，您可能会为网络中的核心交换机、分布式交换机和接入交换机创建角色。

## 字段

### 名称

一个独特的、人类友好的名称。

### Slug

一个独特的、URL友好的标识符。（此值可用于过滤。）

### 颜色

在NetBox UI中显示角色时使用的颜色。

### VM角色

如果选中，此角色可以分配给[虚拟机](../virtualization/virtualmachine.md)

### 配置模板

分配给此角色的设备的默认[配置模板](../extras/configtemplate.md)。
